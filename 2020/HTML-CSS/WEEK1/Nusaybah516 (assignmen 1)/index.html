Developer Student Club (DSC)

Student Clubs are university based community groups for students interested
in Google developer technologies. Students from all undergraduate or programs
with an interest in growing as a developer are welcome. By joining a DSC,
students grow thier knowledge in a peer-to-peer learning environment and build
solutions for local businessess and their community.

What we offer you:

  • Game development
  • Mobile development
  • Frontend Web development
  • Python
  • UI
  • Cyber security
